# AdTruck — анализ рынка, стоимости разработки и цены продажи

> Отчёт по deep-research (Польша/Европа, состояние на 2025–2026). Собран из двух прогонов
> исследования с состязательной проверкой фактов (3 голоса на утверждение, ≥2 refute «убивают»).
> Первый прогон (конкуренты) синтезирован полностью. Второй (ставки/цены) успел проверить
> ставки разработчиков; проверка SaaS-прайсов и MVP-бюджетов прервана лимитом сессии —
> эти цифры взяты **дословно с первоисточников** (прайс-страницы вендоров, отчёты No Fluff
> Jobs / justjoin.it), но их состязательная верификация не завершена. Помечено в тексте.

## Резюме для решения

1. **GPS-трекинг рекламной кампании на карте — не «фишка», а гигиенический минимум зрелого mobile-OOH рынка.** В США/Канаде это есть у Movia, Wrapify (патент), исторического инкумбента TRUCK ADS и Mobilads ещё с 2018 г. Технически AdTruck не изобретает ничего нового.
2. **Но в Польше ниша «живой трекинг кампании для рекламодателя» выглядит свободной.** Единственный найденный локальный игрок в truck-рекламе — брокер **TIR ADS** (Warszawska Grupa Mediowa): без GPS, карты и клиентского дашборда. Европейских аналогов уровня Movia/Wrapify исследование не нашло. Дифференциатор AdTruck — не сам GPS, а **связка «двусторонний маркетплейс (рекламодатель ↔ автопарк) + польский рынок + продуктовый UX уровня Apple»**.
3. **GPS-слой рационально купить, а не строить.** Есть готовые white-label-платформы (Wialon/Gurtam с SDK и открытым API, GPSWOX с публичными ценами, Navixy). Для POC мы всё равно делаем симулятор, но в MVP решение «buy vs build» экономит месяцы.
4. **Стоимость постройки в Польше умеренная.** Реалистичный бюджет MVP силами польского софтхауса — **$18–40 тыс.** (≈ 70–160 тыс. zł), 3–5 месяцев. Blended-ставка агентства — $45–99/ч; себестоимость senior-контрактора по локальным данным — ~130–165 PLN/ч.
5. **Готовность малого бизнеса платить за трекинг — низкая, ~20–60 zł/машина/мес.** Это бенчмарк для «трекинговой» части ценностного предложения; рекламная часть (аренда борта, ~3 000 zł/naczepa/мес в нашей модели) — принципиально другой, гораздо более ёмкий чек.

## Часть 1. Конкуренты

### 1.1. Truck-side / mobile-OOH платформы (глобальные)

| Платформа | Что это | GPS/трекинг | Модель / цены | Уверенность |
|---|---|---|---|---|
| **Movia Media** (HaulerAds, US/CA) | Реклама на бортах грузовиков как managed-сервис | Live-GPS + Wi-Fi/Bluetooth-подсчёт показов, дашборд рекламодателя | $300–700/грузовик/день (статик), $700–1 800 (digital) | high (3-0) |
| **Wrapify** (US) | Mobile-OOH на легковых (райдшеринг), 400 000+ водителей | **Запатентованный** (US10373205B2) GPS→impressions-дашборд + атрибуция | Водителям «сотни $/мес»; рекламодателям — цены по запросу | high (3-0) |
| **Momentara** (ex-AllOver Media, US) | Truckside managed-обклейка | «Fleet Management Integration» (in-house), proof-of-performance; **явных упоминаний GPS/live-карты нет** | Цены по запросу; гео-таргет в терминах US (zip/DMA) | high (3-0 / 2-1) |
| **TRUCK ADS** (US, с 1996; куплен Nickelytics/Robot.com 09.2024) | Инкумбент truck-рекламы | «GPS в каждом грузовике», web-трекинг, ежедневный proof-of-performance, «GRP Verified by GPS» (DEC/CPM/2X-фактор) | Долларовых тарифов не публиковал | high (3-0) |
| **Mobilads** (NYC) | Mobile-OOH на райдшеринге | GPS каждые 10 сек + mobile-ID crowd-аналитика (с 2018) | — | medium (3-0, вторичный источник) |

**Вывод:** функционал AdTruck на глобальном рынке давно существует и стандартен. Ни один из этих игроков не работает в Польше и не является self-serve двусторонним маркетплейсом на польском языке.

### 1.2. Локальный конкурент в Польше

- **TIR ADS** (Warszawska Grupa Mediowa, Варшава, KRS 847360) — брокер рекламы на грузовиках без своего автопарка. В оффере есть «Monitoring tras i sprawozdawczość», **но ни одного упоминания GPS, карты (mapa), śledzenia или клиентского панели** — мониторинг похож на ручную агентскую отчётность. Цены не публикуются. *(high, 3-0; негативная часть проверена подсчётом ключевых слов по HTML.)*
- **Технологическая ниша AdTruck (живой GPS-трекинг кампании для рекламодателя) в Польше этим игроком не закрыта.**

### 1.3. White-label GPS как «купить вместо построить»

| Платформа | White-label | API/SDK | Публичные цены | Уверенность |
|---|---|---|---|---|
| **Wialon (Gurtam)** | Да (лого, интерфейсы, домен, мобильные приложения) | **SDK + открытый Remote API** (sdk.wialon.com, help.wialon.com/en/api) — можно построить слой карты кампаний поверх | Нет; только через партнёров (sales@wialon.com) | high (3-0) |
| **GPSWOX** | Да (свой домен/лого, биллинг клиентов, API) | Есть | **Единственный с открытыми ценами:** hosted Pro €49/мес (1–50 объектов ≈ €1/машина/мес), White Label от €99/мес (1–100); безлимит — только lifetime $9 900 единовременно + поддержка 49 EUR/мес. Брендированные приложения — платные аддоны ($495–795 разово) | high (3-0; маркетинговый «unlimited от $99» опровергнут 0-3) |
| **Navixy** | Да (multi-tenant для TSP/реселлеров, «50%+ margins») | Есть | Usage-based «как AWS/GCP», цены по запросу | high (3-0) |

**Вывод:** для MVP реально взять Wialon/GPSWOX как трекинг-бэкенд и строить поверх только продуктовый слой (маркетплейс, кампании, UX). Это меняет оценку сроков MVP в меньшую сторону.

|---|---|---|

| MVP «quality» (general) | $8 000–30 000 (5hz) | — |

## Выводы для AdTruck

- **Позиционирование:** не «ещё один GPS-трекер», а **маркетплейс рекламы на фурах с прозрачным для рекламодателя live-подтверждением охвата**, локализованный под Польшу. Дифференциаторы: двусторонность, польский рынок, продуктовый UX.
- **Свободная ниша** в PL подтверждается: локальный конкурент — брокер без технологии.
- **Build vs buy:** GPS-бэкенд в MVP брать готовый (Wialon SDK / GPSWOX white-label), силы команды — на маркетплейс и UX.
- **Бюджет разработки MVP:** ориентир **$18–40 тыс. / 70–160 тыс. zł, 3–5 мес**.
- **Монетизация:** ориентироваться на рекламный чек (аренда бортов), а не на телематические 20–60 zł/машина; трекинг — часть value proposition, не основной источник дохода.

## Оговорки и достоверность

- **Проверено состязательно (high):** вся Часть 1 (конкуренты), ставки разработчиков justjoin.it/No Fluff Jobs (Часть 2.1).
- **Первоисточник, но верификация прервана лимитом сессии** (цифры «как на странице», перепроверить перед КП): MVP-бюджеты (2.2), непрограммистские ставки, все польские SaaS-прайсы (Часть 3). Это инфраструктурный обрыв, не опровержение — источники первичные, цитаты дословные.
- **Опровергнуто (для прозрачности):** «GPSWOX — безлимит объектов от $99/мес» (0-3): помесячный тир $99 ограничен 100 объектами, безлимит — только lifetime $9 900.
- **Не покрыто:** Webfleet (назван в исходном вопросе), планы Nickelytics/Robot.com по выходу в ЕС, экономика стороны перевозчика (сколько платформы платят автопаркам).
- **Времязависимость:** живые страницы проверялись 01–02.07.2026; промо GPSWOX — до 31.07.2026; TRUCK ADS — исторический бенчмарк (по Wayback, URL редиректит на robot.com/ads).
