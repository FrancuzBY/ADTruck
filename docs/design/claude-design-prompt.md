# AdTruck — высокоточный кликабельный прототип мобильного приложения

Собери **один самодостаточный интерактивный прототип** мобильного приложения AdTruck: **11 экранов** (10 продуктовых + 1 маркетинговая обложка), внутри реалистичного **iPhone-фрейма (390×844, iPhone 15)**, с **рабочей навигацией между экранами** и оркестрированной анимацией. Это питч-артефакт для клиента, а не вайрфрейм — каждый экран должен выглядеть как отгруженный премиум-продукт.

## 0. Что это за продукт (чтобы решения были осмысленными)

**AdTruck** — двусторонняя платформа: **реклама на бортах фур TIR (полуприцепов-naczep), которые едут через всю Европу**, плюс **live-карта**, где видно, где прямо сейчас находится каждая фура (а значит — каждый рекламный баннер). Рынок — **Польша**, весь UI на польском. Mobile-first PWA.

Две роли (переключаются в Профиле, меняя 3-ю вкладку нижней навигации):
- **Reklamodawca** (рекламодатель) — заказывает кампании, смотрит охват.
- **Przewoźnik** (перевозчик) — управляет своим автопарком naczep, зарабатывает на сдаче бортов.

**Эмоциональное ядро продукта:** «*Twoja reklama jedzie przez całą Europę*» — твоя реклама физически едет по Европе, и за ней можно следить вживую. Весь дизайн должен это чувство усиливать.

## 1. Визуальное направление — ГИБРИД из двух режимов одной системы

Это не два разных приложения — это **один продукт с двумя настроениями**, как iOS: светлый чистый интерфейс приложения + тёмная иммерсивная Карта. Общие: типографика, сетка, язык движения, оттенок акцента.

- **Режим A — «Keynote» (СВЕТЛЫЙ):** обложка, лендинг, весь визард заказа (3 шага), список кампаний, форма добавления фуры, профиль. Спокойно, премиально, много воздуха, доверие. Это поверхность **покупки и маркетинга**.
- **Режим B — «Mission Control» (ТЁМНЫЙ):** live-карта, отчёт кампании (дашборд), список флоты. Иммерсивно, данные, телеметрия, **карта светится**. Это поверхность «**смотри, как твоя реклама едет**» и операционка перевозчика — главный «вау».

Ритм показа специально контрастный: светлый поток покупки → *провал в тёмный* mission-control, когда реклама «поехала». Этот переход — драматический момент питча.

## 2. Дизайн-система (токены — задай через CSS-переменные)

### Цвет

**Light (Keynote):**
```
--ink        #0B1220   /* заголовки, основной текст */
--ink-muted  #5B6472   /* вторичный текст */
--canvas     #F5F7FB   /* фон полотна (не чисто белый) */
--surface    #FFFFFF   /* карточки */
--line       #E6EAF2   /* хайрлайн-границы */
--brand      #1E3A8A   /* глубокий navy — бренд */
--accent     #3B82F6   /* единственный акцент (ссылки, активное) */
--cta        #16A34A   /* зелёный CTA (заказ, деньги, успех) */
--cta-strong #15803D
```

**Dark (Mission Control):**
```
--base       #0A0E14   /* почти-чёрный фон */
--panel      #121826   /* панели/карточки */
--panel-2    #1A2233   /* приподнятые панели */
--hair       #232C3D   /* границы в темноте */
--text       #E7ECF5
--text-mut   #8A94A6
--neon       #22D3EE   /* cyan — акцент телеметрии, выделение */
--live       #A3E635   /* лайм — «в движении» / driving */
--warn       #F59E0B   /* «postój» / внимание */
```

Правило: **доминирующий фон + один резкий акцент**. Никаких «радужных» равномерных палитр. Зелёный = только деньги/CTA/успех. Cyan/лайм = только на тёмных экранах телеметрии.

Статусы кампании (verbatim, светлый режим): `Zaplanowana` (янтарный), `Aktywna` (зелёный), `Zakończona` (серый).

### Типографика

Шрифты **обязаны поддерживать польские диакритики** (ą ć ę ł ń ó ś ź ż). Не используй Inter/Roboto/Arial/системный как основной.
- **Display (заголовки, крупные цифры):** `Bricolage Grotesque` — характерный, премиальный.
- **Body / UI:** `Geist` — чистый, современный.
- **Numeric / телеметрия:** `Geist Mono` или `JetBrains Mono`, всегда `font-variant-numeric: tabular-nums` для всех чисел, км/ч, цен, процентов.

Подключи через Google Fonts, но **всегда дай надёжный fallback-стек** `-apple-system, "SF Pro Display", "Segoe UI", sans-serif`, чтобы при блокировке веб-шрифтов всё равно читалось «по-эпловски». Заголовки крупные и уверенные (32–44 на экранах, 56–72 на обложке), тугой трекинг у крупного; тело 15–17; подписи 12–13. Контраст размеров — часть драмы.

### Пространство, форма, глубина

- База 4px. **Щедрый воздух** в светлом режиме; контролируемая плотность в тёмном.
- Радиусы: карточки 20–24px, кнопки — pill (полностью круглые), инпуты 14–16px.
- Свет: мягкая многослойная тень `0 1px 2px rgb(15 23 42/.06), 0 12px 32px -16px rgb(15 23 42/.14)`. Тёмный: хайрлайн-границы + свечение (glow) вместо теней.
- Атмосфера, не плоская заливка: свет — едва заметный градиент/зерно на полотне, мягкое «свечение покрытия» под мини-картой; тёмный — радиальный виньет, тонкая сетка-грид, **светящиеся линии маршрутов**, лёгкий блюр под картой.

### Иконки и движение

- Иконки — тонкая линия (stroke 1.75–2px) на сетке 24px, единый набор: дом, карта, мегафон, фура, юзер. Никаких эмодзи вместо иконок.
- Движение (Motion/framer-motion, 60fps): вход экрана — **стаггер-появление** блоков (y+opacity, 60ms шаг); тап по кнопке — spring scale 0.97; **числа считаются count-up** (цена, км, показы, проценты); галочка успеха — **path-draw**; заполнение слайдера — тактильное, thumb увеличивается при захвате; на карте — **фуры едут** вдоль polyline, линии маршрутов **прорисовываются** при загрузке; прогресс-бар и бары покрытия растут слева направо. Один хорошо поставленный момент на экран важнее десятка мелких.

### Устройство

Показывай экраны в реалистичном iPhone-фрейме: скруглённые углы, тонкая рамка, **status bar** (польское время `9:41`, оператор, Wi-Fi, батарея), home-indicator. Внизу — рабочая **нижняя навигация**: `Start` · `Mapa` · `Kampanie` (или `Flota` в роли перевозчика) · `Profil`. Активная вкладка — акцентом.

## 3. Экраны (11 штук) — верстай каждый с реальным контентом и данными

Числа ниже **внутренне согласованы** (это реальная модель приложения): фикстура `50 naczep × 3 000 zł × 3 miesiące = 450 000 zł`. Формат чисел польский: неразрывный пробел разрядов, запятая-десятичная, `zł` в конце, `mln` для миллионов, даты вида `1 maj 2026`.

### (0) Обложка питча — «Hero cover» · СВЕТЛЫЙ (или драматический тёмный — на твоё усмотрение, что эффектнее)
Открывающий кадр. Крупно: `AdTruck` и тэглайн **`Reklama, która jedzie przez Europę`**. Рядом — iPhone с live-картой Европы, по которой светящейся линией идёт маршрут и едет фура. Стилизованная Европа, ощущение движения и масштаба. Это обложка для первого слайда — должна вызывать «вау» за 1 секунду.

### (1) Лендинг · `/` · СВЕТЛЫЙ
- Hero-блок (навигация-бренд сверху): eyebrow `AdTruck`, заголовок **`Twoja reklama jedzie przez całą Europę`**, подзаголовок `Wynajmij powierzchnię reklamową na naczepach TIR i śledź zasięg swojej kampanii na żywo.`, зелёный pill-CTA **`Zamów kampanię`**. За/под текстом — тонкая анимированная линия маршрута по мини-Европе.
- Три карточки-преимущества (стаггер при входе):
  - `GPS na żywo` — `Każda naczepa z Twoją reklamą widoczna na mapie w czasie rzeczywistym.`
  - `Zasięg w całej Europie` — `Trasy przez 20+ krajów — od Warszawy po Madryt.`
  - `Raporty i statystyki` — `Przebieg, wyświetlenia i pokrycie kampanii w przejrzystych raportach.`
- Тап по CTA ведёт на экран (2).

### (2) Визард шаг 1 — Заказ · `/zamow` · СВЕТЛЫЙ
Топбар: кнопка «назад», заголовок `Zamów kampanię`, индикатор прогресса из 3 точек (шаг 1 активен).
- Карточка `Termin kampanii`: два поля-даты `Od` = `1 maj 2026`, `Do` = `31 lip 2026`.
- Карточка `Zasięg w Europie`: мини-карта Европы с несколькими светящимися маршрутами (превью охвата, glow под ней).
- Карточка с двумя **тактильными слайдерами** (залитый трек, крупный thumb, живой пересчёт):
  - `Liczba naczep` — значение `50 naczepy` (диапазон 10–100).
  - `Budżet miesięczny` — значение `150 000 zł` (диапазон 30 000–400 000 zł), подпись-хинт `Przy tym budżecie: do 50 naczep`.
- Зелёный CTA `Dalej` → экран (3).

### (3) Визард шаг 2 — Итог · `/zamow/podsumowanie` · СВЕТЛЫЙ · ⭐ денежный момент
Топбар `Podsumowanie`, прогресс 2/3. Карточка-список (StatRow, tabular-nums):
- `Czas trwania` → `3 miesiące`
- `Liczba naczep` → `50 naczep`
- `Szacowany przebieg` → `1 500 000 km`
- `Zasięg (wyświetlenia)` → `4,7–8,8 mln`

Отдельная **акцентная зелёная карточка** (лёгкий зелёный фон):
- Крупно `Koszt kampanii` → **`450 000 zł`** (шрифт 28–32, зелёный `--cta-strong`, **count-up при появлении**).
- Мелко под ним: `50 × 3 000 zł × 3 miesiące`.

Дисклеймер мелким серым: `Zasięg szacowany na podstawie przebiegu i średniej liczby kontaktów wzrokowych (OTS) na trasach miejskich i tranzytowych. Zakres ±30%.`
Зелёный CTA `Zamawiam` → экран (4).

### (4) Визард шаг 3 — Подтверждение · `/zamow/potwierdzenie` · СВЕТЛЫЙ
По центру: зелёный круг с **рисующейся галочкой** (spring). Заголовок `Kampania utworzona!`, подзаголовок `Twoja reklama wkrótce ruszy w trasę. Poniżej znajdziesz narzędzia do śledzenia.`
Две карточки-действия (стаггер):
- `GPS na żywo` — `Śledź swoje naczepy na mapie Europy w czasie rzeczywistym.` → кнопка `Zobacz mapę` (ведёт на экран 5).
- `Raporty kampanii` — `Przebieg, wyświetlenia i pokrycie w jednym miejscu.` → кнопка `Twoje kampanie` (экран 6).
Внизу ghost-кнопка `Wróć na stronę główną`.

### (5) Live-карта · `/mapa` · ТЁМНЫЙ · ⭐⭐ главный «вау»
**Карта Европы во весь экран** (тёмная, иммерсивная): страны едва различимы, **светящиеся линии маршрутов**, по ним **едут десятки фур** — точки-«живчики» с лёгким шлейфом/glow, driving подсвечены лаймом. Радиальный виньет к краям.
- Сверху слева: pill `Mapa na żywo` + переключатель скорости-pill `1×` / `60×` (60× — фуры едут заметно быстрее).
- Счётчик-HUD: `127 naczep w trasie` (count-up).
- Внизу — приподнятая HUD-карточка выбранной фуры: `WGM 8842` · `Warszawa – Madryt` · `82 km/h` · `Niemcy`, мини-спарклайн скорости.
Это экран, ради которого всё затевалось — он должен выглядеть как пульт управления флотом уровня Flighty/Linear.

### (6) Список кампаний · `/kampanie` · СВЕТЛЫЙ
Заголовок `Twoje kampanie`. Карточки кампаний (стаггер, hover-lift), каждая:
- строка дат `1 maj 2026 – 31 lip 2026` + бейдж статуса `Aktywna`;
- нижняя строка: `50 naczep` (слева, серым) и `450 000 zł` (справа, жирным).
Примеры карточек: `Aktywna` (450 000 zł), `Zaplanowana` (1 sie 2026 – 30 wrz 2026, 30 naczep, 270 000 zł), `Zakończona` (1 lut 2026 – 30 kwi 2026, 40 naczep, 360 000 zł).
Внизу вторичная кнопка `Nowa kampania`.
*(Также покажи пустое состояние как вариант: карточка `Nie masz jeszcze żadnej kampanii.` + зелёная кнопка `Zamów pierwszą kampanię`.)*
Тап по карточке → экран (7).

### (7) Отчёт кампании (дашборд) · `/kampanie/:id` · ТЁМНЫЙ · ⭐ дашборд

- Карточка-шапка: `1 maj 2026 – 31 lip 2026` + бейдж `Aktywna`; тонкий **прогресс-бар** (заполняется) и подпись `Postęp kampanii: 42%`.
- Карточка-метрики (tabular-nums, count-up): `Naczepy` → `50 naczep` · `Przebieg (dotychczas)` → `631 000 km` · `Wyświetlenia` → `2,0–3,7 mln` · `Kraje` → `7`.
- Карточка `Pokrycie wg krajów` — горизонтальные бары (растут слева направо), польские названия стран + число фур:
  `Polska` 18 · `Niemcy` 11 · `Czechy` 7 · `Austria` 5 · `Francja` 4 · `Holandia` 3 · `Włochy` 2. (Бар нормируется к максимуму = 18.)
- Карточка `Naczepy kampanii na żywo` — встроенная тёмная мини-карта с едущими фурами этой кампании.
- Пунктирная карточка-заглушка загрузки креатива: `Kreacja reklamowa` / `Wgrywanie plików — wkrótce`.

- `KR 9021G` · `Kraków – Paryż` · `postój` · `289 km dziś` · стоит
- `GD 7788` · `Gdańsk – Rzym` · `91 km/h` · `503 km dziś` · едет
- `WR 302AK` · `Wrocław – Amsterdam` · `74 km/h` · `268 km dziś` · едет

### (10) Профиль · `/profil` · СВЕТЛЫЙ
Заголовок `Profil`. Три секции-карточки:
- `Rola w demo` — сегмент-переключатель из двух pill: `Reklamodawca` / `Przewoźnik` (активный залит брендом; при переключении pill плавно скользит). Смена роли меняет 3-ю вкладку низа (`Kampanie` ↔ `Flota`).
- `Prędkość symulacji` — две pill `1×` / `60×`.
- Карточка-подсказка: `Dodaj AdTruck do ekranu głównego, aby korzystać jak z aplikacji.` + кнопка-контур `Resetuj demo`.

## 4. Чего НЕ делать (анти-slop)

- ❌ Inter/Roboto/Arial/системный как основной шрифт; фиолетовые градиенты на белом; любые generic-AI-паттерны.
- ❌ Lorem ipsum или английский в UI — только польские строки выше, verbatim, с диакритиками.
- ❌ Плоские серые вайрфрейм-карточки, равномерная «радужная» палитра, Bootstrap/Material-вид.
- ❌ Эмодзи вместо иконок, сток-клипарт, случайные числа, противоречащие модели цен.
- ❌ Всё по центру и одинаковыми блоками. Нужна иерархия, контраст размеров, воздух, ритм.
- ❌ Горизонтальный скролл; широкий контент (карта, бары) скроллится/помещается внутри своего контейнера.

## 5. Формат результата

- **Один самодостаточный файл** (HTML+CSS+JS инлайн, либо React-артефакт), всё встроено; внешние ресурсы — максимум Google Fonts со стек-fallback.
- iPhone-фрейм 390×844. **Рабочая навигация**: тапы по CTA и нижней навигации переключают экраны (можно «пройти» сценарий: Лендинг → Zamów → Podsumowanie → Zamawiam → успех → Zobacz mapę → тёмная карта; и роль-свитч Reklamodawca↔Przewoźnik меняет Kampanie↔Flota).
- Плюс дай удобный **переключатель экранов** (сетка-галерея всех 11 фреймов), чтобы клиенту показать всё сразу.
- 60fps анимации, корректный рендер польских диакритик, никакого горизонтального скролла, идеальные отступы и типографика. Планка — отгружаемый премиум-продукт.
